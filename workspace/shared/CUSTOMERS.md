# CUSTOMERS.md — Nanira Customer Database

> Customer relationship database for **Nanira Family Reflexology**. Maintained primarily by **OKI (COO)** with retention support. Read by Reyns for reporting. Keep phone numbers in international format where possible (+62…).

## Retention Status legend
- 🟢 **Active** — visited within the last 30 days.
- 🟡 **At Risk** — 30–60 days since last visit (Oki sends a personal WhatsApp message).
- 🔴 **Dormant** — > 60 days since last visit (re-engagement campaign).
- ⭐ **VIP** — high-frequency / high-spend regular.
- 🆕 **New** — first visit logged, no return yet.

## Retention workflow (Oki)
1. After every visit, update **Last Visit** and **Favorite Treatment**.
2. If no return within **30 days** → send personal WhatsApp message, set status 🟡.
3. Log preferences, allergies, therapist preference, and special notes.
4. Escalate to Reyns: refund requests, named-staff complaints, angry customers.
