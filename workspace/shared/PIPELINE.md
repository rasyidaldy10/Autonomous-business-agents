# PIPELINE.md — Golden Medical Sales Pipeline

> Sales pipeline for **PT Golden Medical Care Indonesia**. **RIKO (Sales Manager) is the SOLE OWNER of lead status updates** — no other agent edits status to prevent conflicting versions. Nabil feeds in LKPP tenders (with estimated value); Anggi feeds in inbound leads; Riko tracks and prioritises.

## Status flow
`New → Pending Info → Draft → Review → Sent → Follow Up → Won / Lost`

## Priority Score
**Priority Score = Project Value (Rp) × Closing Probability (%)**
Follow-ups are worked in **descending Priority Score order — NOT by date received.**

## Escalation tags (Riko → Reyns)
- Lead > **Rp100M** → **CRITICAL / HIGH** tag.
- Tender deadline < **7 days** → **HIGH** tag.
- Proposal **Sent** but not followed up > **3 days** → reminder.

## Lost reason
Always record a **Lost reason** when a lead is marked Lost (for analysis: price, timeline, competitor, no budget, no response, etc.).
